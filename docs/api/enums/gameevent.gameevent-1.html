Enumeration GameEvent

GameEvents describe things that happen in the game world. Generally, if this actions are motivated by a player's verb, handlers for these events will be called after the verb "pre" phase, but before the verb "post" phase.

If there are global listeners and per-instance listeners, global listeners are invoked first, then per-instance listeners.
